# Hahow backend recruitment

## Requirements
 - [nvm](https://github.com/nvm-sh/nvm) 0.39.3+
 - [Node.js](https://nodejs.org/) 18+
 - [npm](https://www.npmjs.com/) (normally comes with Node.js)

## How to start
1. Clone repository
2. Install package
    ```
    npm install
    ```
## How to use    
 -  Start API server
    ```
    npm start
    ```
 -  Test for APIs
    ```
    npm test
    ```
